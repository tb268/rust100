# Rust 100日チャレンジ 🦀

Rustを100日間学習する記録

## 進捗 (3/100)

| Day | プロジェクト | 内容 | 学んだこと |
|-----|------------|------|-----------|
| 001 | [hello-rust](./001/hello-rust) | Hello World | 基本構文、cargo の使い方 |
| 002 | [number](./002/number) | 数値計算 | 変数、型、標準入力 |
| 003 | [fizzbuzz](./fizzbuzz_003) | FizzBuzz | 条件分岐、ループ |

## 目標
- Rustの基礎を習得
- 毎日コミット
- 実践的なプロジェクトを作る

## 環境
- Rust: stable
- OS: macOS

## 実行方法

各プロジェクトのディレクトリに移動して実行：

```bash
cd 001/hello-rust
cargo run
```
